# Одиссея Гомера — разбор по песням

Маленький **статический сайт**: краткое описание сюжета и главных моментов поэмы Гомера
«Одиссея» по всем 24 песням. Отдельный проект, никак не связанный с другими репозиториями.

Стек намеренно простой — **без Flask, без сервера, без зависимостей**: чистый Python-генератор
собирает HTML, который можно открыть локально или бесплатно захостить как статику.

**Живой сайт:** https://rin900work-png.github.io/odyssey-of-homer/

## Структура

```
content.py        — все данные (текст разбора): мета, 3 части, герои, темы, 24 песни
build.py          — генератор: читает content.py → собирает HTML в docs/
static/style.css  — стили (тема «эгейское море»: пергамент + синь)
docs/             — СГЕНЕРИРОВАННЫЙ сайт (index.html, song-01..24.html, style.css)
```

Хочешь поправить текст разбора — правишь `content.py` и пересобираешь. Менять HTML руками
не нужно.

## Сборка

Нужен только Python 3 (стандартная библиотека, никаких `pip install`):

```bash
python3 build.py
```

Команда заново собирает папку `docs/`. Посмотреть локально — просто открыть
`docs/index.html` в браузере.

## Композиция (3 части)

- **I. Телемахия** (песни 1–4) — Итака без Одиссея, путешествие Телемаха.
- **II. Странствия** (песни 5–12) — феаки и рассказ Одиссея: киклоп, Цирцея, царство мёртвых, сирены, Сцилла и Харибда.
- **III. Возвращение и месть** (песни 13–24) — узнавания, лук, расправа над женихами, примирение.

## Как опубликовать (GitHub Pages)

Сайт уже раздаётся через GitHub Pages из папки `docs/` ветки `main`.

Чтобы обновить сайт после правок:

```bash
python3 build.py        # пересобрать docs/
git add -A
git commit -m "обновил разбор"
git push                # GitHub Pages подхватит изменения за ~минуту
```

## Другие варианты хостинга

Сайт — статика, сервер не нужен. Папку `docs/` можно так же раздать на Cloudflare Pages,
Netlify или Vercel. Понадобится бэкенд — брать serverless (Cloudflare Workers / Vercel
functions), а не выделенный сервер.
