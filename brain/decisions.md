# Архитектурные решения — Одиссея Гомера

### Статический генератор на чистом Python (без Flask, без зависимостей)
Почему: сайт — это статика, сервер не нужен; проще и бесплатно хостить на GitHub Pages.
Отвергли: Flask/сервер (избыточно для статики), Jekyll/SSG-фреймворк (лишние зависимости и
магия сборки — хотелось голый `python3 build.py`).

### Контент отделён от вёрстки: content.py → build.py → docs/
Почему: править текст разбора, не трогая HTML; единый каркас страниц в одном месте (`layout()`).
`docs/` — генерируемая папка (build делает `rmtree` + пересборку), руками не редактируется.

### Хостинг — GitHub Pages из папки docs/ ветки main
`.nojekyll` (пишется build-ом) отключает обработку Jekyll — файлы раздаются как есть.
Push в `main` → деплой за ~минуту. При необходимости бэкенда — только serverless, не VPS.
